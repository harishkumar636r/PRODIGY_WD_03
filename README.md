Description

This is a modern and responsive Tic-Tac-Toe web application built using HTML, CSS, and JavaScript.
It allows two players to compete in a classic game of Tic-Tac-Toe. The app tracks the game state, detects wins and draws, and provides a restart button to start a new game.
With color-coded X and O, hover animations, and a stylish glassmorphism UI, this game offers a clean and engaging user experience.

Two-Player Gameplay: Play Tic-Tac-Toe with a friend locally.

Win & Draw Detection: Automatically detects winning combinations and draws.

Restart Game Button: Quickly reset the board to start a new match.

Modern UI: Stylish glassmorphism-inspired design for a sleek look.

Hover Animations: Interactive hover effects on each cell for better UX.

Responsive Design: Works smoothly on both desktop and mobile devices.

Color-Coded Symbols: Distinct colors for X and O for easy distinction.

Interactive Notifications: Highlights the winner and indicates draw scenarios.

How to Play

Open the index.html file in your browser.

Players take turns clicking on empty cells to place their symbol (X or O).
